HealthNest authentication redesign
==================================

Updated pages:
- login.php
- register.php
- assets/style.css
- assets/auth.js (new)

Included improvements:
- Reference-inspired dark glass login and registration design
- Home buttons on both pages
- High-contrast visible input text and placeholders
- Show/Hide controls for login, password, and confirm-password fields
- Buyer/Seller registration dropdown with server-side validation
- 18+ date limit with live feedback and server-side enforcement
- Immediate password-match validation plus existing server-side validation
- Responsive mobile layout and accessible field labels
- Compact, better-balanced card and input sizing
- Custom keyboard-accessible Buyer/Seller dropdown
- Original HealthNest dark blue and green visual palette

To use with XAMPP:
1. Extract the try1 folder into C:\xampp\htdocs\
2. Start Apache and MySQL in XAMPP.
3. Import db_healthnest.sql if the db_healthnest database is not installed yet.
4. Open http://localhost/try1/
